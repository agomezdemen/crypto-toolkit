# Crypto Toolkit (Rust) - Educational
**This project is for learning and demonstration purposes. Do not use in production.**

## Features (v0.1)
- RSA (keygen, OAEP encrypt/decrypt, PSS sign/verify)
- AES-256-GCM (AEAD)
- SHA-256 / BLAKE3
- PEM import/export
- CLI with 'clap'
